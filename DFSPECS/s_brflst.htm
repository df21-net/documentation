BRIEFING.LST

In DARK.GOB we have the file BRIEFINGS.LST. It contains a list of mission briefings, and its contents should correspond to the levels found in JEDI.LVL:

Header

BRF 1.0
BRIEFS 14

Contents

After the header follows lines containing the information for each level's briefing. Each line contains: 

LEV:   name of the level
LFD:   name of the .LFD file to take the briefing from
ANI:   name of the ANIM (in the .LFD file) to use as background
PAL:   the palette to use

[It seems that LFD containing briefings and objective screens must be called DFBRIEF.LFD, or the PDA won't work properly - Jereth]
