DARK.EXE Command Line Parameters
Here are all the command line parameters for the Dark Forces executable:
(ordered by probable usefulness)

Parameter Description
-ugobuse an user gob file (where gob=your gob file)
-shotsenable screen shot mode (use Print Screen key)
-cdisable cutscenes
-llevelplay a particular level (where level=yourlevel)
-xdspecify CD-ROM drive letter (where d=drive)
-fdon't check to see if FILES= is set high enough
-tautotest mode (runs all levels briefly)
-gcreate a text file with a list of all files that were opened during the running of the game
