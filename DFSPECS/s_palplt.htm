PAL Files

PAL files are 768 bytes long, and store a single 256 colors palette as 256 x 3 RGB bytes.

Each level in Dark Forces can have its own palette, specified in the LEV header. 

PAL_File IS
{
 Colors           RGB_Color[256]   // 256 colors from 0 to 255
}

   



Where:




RGB_Color Is
{
 R                byte             // Red   intensity
 G                byte             // Green intensity
 B                byte             // Blue  intensity
}

   



Note

The intensities range from 0 to 63 (limit of VGA mode 0x13) in the PAL files.


    Further information



    
    
  • PAL color 0 is treated as transparent where relevant.
    • 
    
  • Colors 1-23 are set as fully bright under all illumination levels in the 
    Dark Forces CMP files. They are used to create parts of
    textures and sprites that "glow in the dark".
    • 
    
  • Colours 24-31 are specially reserved for the HUD health, shield and battery displays.
    • 
    
  • Colors 32-207 are "universal" for all Dark Forces PALs. Textures and sprites using
    these colors can be used with every PAL without distortion.
    • 
    
  • Colors 208-255 are different from one PAL to another. They are used for 
    textures that are specific to a certain level (eg. Nar Shaddaa, Robotics) and 
    will look "wrong" if used in other levels. For example, the Kell Dragon's 
    distorted appearance if placed in SECBASE.
    • 









PLTT Resources (PLT Files)


PLTT LFD resources are of variable size, and store a (possibly incomplete) palette used by ANIM and DELT resources.



Note

The .plt extension is a convention adopted by add-on developers when writing conversion programs, there are no real PLT files in DARK FORCES.




PLTT_File IS
{
 First            byte             // first color in the palette
 Last             byte             // last color in the palette
 colors           RGB_Color[n]     // n = Last - First + 1
 pad1             byte             // unused = 0
}

   



Where:




RGB_Color Is
{
 R                byte             // Red   intensity
 G                byte             // Green intensity
 B                byte             // Blue  intensity
}

   



Note

Contrary to the PAL files, the intensities range from 0 to 255 in the PLTT resources.
